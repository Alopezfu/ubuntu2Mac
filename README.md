# ubuntu2Mac
Ubuntu desktop like Mac Desktop (GNOME)

You only need run this command in ubnutu2mac folder: 

- git clone https://github.com/Alopezfu/ubuntu2Mac.git

- cd ubuntu2Mac

- chmod +x install

- ./install

***AND Enjoy in 5 seconds!***
